所有图片配置都在 `src/App.tsx` 文件中，有两个地方需要修改：

## 1️⃣ 首页大图（4张）

### 位置：第 103-147 行

找到 `HomePage` 函数中的 `sections` 数组：

```typescript

  { 
    type: "image",
    title:"ZM Studio", 
    subtitle:"Exhibition & Architecture", 
    align:"left", 
    url:"这里是图片1的URL" 
  },
  { 
    type: "text",
    heading: "设计哲学", 
    text: "..."
  },
  { 
    type: "image",
    title:"Runway Curve Campus", 
    subtitle:"1.1 km linear → five curve rooms", 
    align:"right", 
    url:"这里是图片2的URL" 
  },
  // ... 更多图片

```

### 🔧 如何替换：

**方法一：使用在线图片**
- 直接替换 `url:` 后面的链接
- 例如：`url:"https://your-image-url.com/photo.jpg"`

**方法二：使用本地图片**

1. **创建图片文件夹**：
   ```
   网站设计/
   └── public/
       └── images/
           ├── hero1.jpg
           ├── hero2.jpg
           ├── hero3.jpg
           └── hero4.jpg
   ```

2. **替换URL**：
   ```typescript
   url:"/images/hero1.jpg"
   ```

## 2️⃣ 项目列表的项目图片（占位符）

### 位置：第14-19行

找到 `PROJECTS` 数组：

```typescript
const PROJECTS = [
  { 
    id: "p1", 
    title: "River Loop Pavilion", 
    coverHue: 200,  // 目前是色块占位符
    gallery: [200,210,220]  // 画廊图片也是色块
  },
  // ... 更多项目
];
```

### 🔧 如何添加真实项目图片：

#### 步骤1：修改项目数据结构

将 `coverHue` 改为 `coverImage`，将 `gallery` 改为图片URL数组：

```typescript
const PROJECTS = [
  { 
    id: "p1", 
    title: "River Loop Pavilion",
    year: 2025,
    location: "Milan, IT",
    typology: "Installation",
    status: "Competition",
    tags: ["Landscape","Pavilion","Curve"],
    coverImage: "/images/projects/p1-cover.jpg",  // 封面图
    summary: "...",
    body: "...",
    gallery: [  // 画廊图片
      "/images/projects/p1-1.jpg",
      "/images/projects/p1-2.jpg",
      "/images/projects/p1-3.jpg"
    ]
  },
];
```

#### 步骤2：修改显示组件

在 `src/App.tsx` 中搜索 `Block` 组件的使用，替换为 `<img>` 标签。

## 📐 图片建议规格

- **首页大图**：建议 2400×1600 像素以上（横向）
- **项目封面**：建议 1200×900 像素（4:3 比例）
- **项目画廊**：建议 1200×900 像素
- **格式**：JPG 或 PNG，建议压缩后单张不超过 500KB

---

## 💡 提示

1. **图片路径说明**：
   - `/images/xxx.jpg` → 从 `public/images/` 文件夹读取
   - `https://xxx.com/xxx.jpg` → 从外部URL读取

2. **保存后自动更新**：
   - 修改代码后保存，浏览器会自动刷新（热更新）
   - 如果添加了新的本地图片到 `public/` 文件夹，需要刷新浏览器

3. **图片对齐方式**：
   - `align:"left"` → 标题在左侧
   - `align:"right"` → 标题在右侧

4. **修改文字内容**：
   - `title` → 大标题
   - `subtitle` → 副标题
   - `heading` → 文字区域的标题
   - `text` → 文字区域的正文

## ❓ 常见问题

**Q: 图片不显示怎么办？**
- 检查图片路径是否正确
- 确保图片文件在 `public/images/` 文件夹中
- 打开浏览器开发者工具（F12）查看错误信息

**Q: 可以用中文文件名吗？**
- 建议使用英文文件名，避免编码问题
- 例如：`hero1.jpg` 而不是 `首页图1.jpg`

**Q: 图片太大加载慢？**
- 使用在线工具压缩图片（如 tinypng.com）
- 或使用图片编辑软件导出为"Web优化"格式
