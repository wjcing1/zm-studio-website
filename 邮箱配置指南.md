## 📮 常用邮箱配置

### ✉️ **Gmail（推荐）**

```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
RECEIVER_EMAIL=your-email@gmail.com
```

**重要：Gmail 需要使用"应用专用密码"**
1. 访问：https://myaccount.google.com/security
2. 开启"两步验证"
3. 进入"应用专用密码"
4. 选择"邮件"和设备
5. 复制生成的16位密码到 `SMTP_PASS`

---

### ✉️ **QQ 邮箱**

```env
SMTP_HOST=smtp.qq.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your-qq-number@qq.com
SMTP_PASS=your-authorization-code
RECEIVER_EMAIL=your-qq-number@qq.com
```

**获取授权码：**
1. 登录 QQ 邮箱网页版
2. 设置 → 账户
3. 开启 SMTP 服务
4. 生成授权码（不是QQ密码！）

---

### ✉️ **163 邮箱**

```env
SMTP_HOST=smtp.163.com
SMTP_PORT=465
SMTP_SECURE=true
SMTP_USER=your-email@163.com
SMTP_PASS=your-authorization-code
RECEIVER_EMAIL=your-email@163.com
```

**获取授权码：**
1. 登录 163 邮箱
2. 设置 → POP3/SMTP/IMAP
3. 开启 SMTP 服务
4. 新增授权密码

---

### ✉️ **Outlook / Hotmail**

```env
SMTP_HOST=smtp-mail.outlook.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your-email@outlook.com
SMTP_PASS=your-password
RECEIVER_EMAIL=your-email@outlook.com
```

## 🌐 Vercel 部署配置

### 在 Vercel 添加环境变量：

1. 进入 Vercel 项目仪表板
2. 点击 **Settings** → **Environment Variables**
3. 添加以下变量：
   - `SMTP_HOST`
   - `SMTP_PORT`
   - `SMTP_SECURE`
   - `SMTP_USER`
   - `SMTP_PASS`
   - `RECEIVER_EMAIL`

4. 点击 **Save**
5. 重新部署项目（Vercel 会自动触发）

## 🔧 常见问题

### ❌ 发送失败？

1. **检查环境变量**
   - 确保 `.env.local` 文件在项目根目录
   - 确保所有变量都已正确填写
   - 密码中没有多余的空格或引号

2. **检查邮箱设置**
   - Gmail: 确认已开启"两步验证"并使用"应用专用密码"
   - QQ/163: 确认已开启 SMTP 服务并使用授权码（不是登录密码）

3. **检查端口和安全设置**
   - 587 端口：`SMTP_SECURE=false`
   - 465 端口：`SMTP_SECURE=true`

4. **防火墙问题**
   - 确保服务器可以访问 SMTP 端口

### ⚠️ 本地能用，部署后不行？

- 检查 Vercel 的环境变量是否正确配置
- 确保重新部署了项目
- 查看 Vercel 的 Functions 日志

## 📝 收到的邮件格式

访客提交表单后，您会收到如下格式的邮件：

**主题：** ZM Studio 网站新消息 - [访客姓名]

**内容：**
```
来自网站的新消息

姓名：张三
邮箱：visitor@example.com
消息内容：
我对你们的展会设计服务很感兴趣...
```

您可以直接点击"回复"来回复访客（因为设置了 replyTo）
