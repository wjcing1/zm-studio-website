# 🚀 联系表单快速配置（3分钟完成）

## 第一步：创建环境变量文件

在项目根目录创建 `.env.local` 文件，内容如下：

```env
# 根据您的邮箱服务商选择配置：

# ========== Gmail 配置 ==========
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=你的邮箱@gmail.com
SMTP_PASS=应用专用密码（见下方获取方法）
RECEIVER_EMAIL=info@zmstudiomilano.it

# ========== QQ邮箱 配置 ==========
# SMTP_HOST=smtp.qq.com
# SMTP_PORT=587
# SMTP_SECURE=false
# SMTP_USER=你的QQ号@qq.com
# SMTP_PASS=QQ邮箱授权码（不是QQ密码！）
# RECEIVER_EMAIL=info@zmstudiomilano.it
```

## 第二步：获取邮箱密码/授权码

### Gmail：
1. 访问：https://myaccount.google.com/security
2. 开启"两步验证"
3. 搜索"应用专用密码"
4. 选择"邮件" → 生成
5. 复制16位密码到 `SMTP_PASS`

### QQ邮箱：
1. 登录 QQ 邮箱网页版
2. 设置 → 账户 → SMTP服务
3. 开启服务 → 生成授权码
4. 复制授权码到 `SMTP_PASS`

## 第三步：本地测试

```bash
npm run dev
```

访问 Contact 页面，填写表单测试。

## 第四步：Vercel 部署配置

1. 登录 Vercel.com
2. 选择项目 → Settings → Environment Variables
3. 添加以下变量（值与 .env.local 中相同）：
   - SMTP_HOST
   - SMTP_PORT
   - SMTP_SECURE
   - SMTP_USER
   - SMTP_PASS
   - RECEIVER_EMAIL

4. 保存后重新部署

## ✅ 完成！

现在访客填写表单后，消息会自动发送到您的邮箱。
