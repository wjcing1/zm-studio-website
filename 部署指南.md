# 🚀 网站部署指南

## 方法一：Vercel 部署（推荐 ⭐）

### 优点：
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN 加速
- ✅ 自动部署（推送代码就更新）
- ✅ 提供免费域名

### 步骤：

#### 1. 安装 Vercel CLI
```bash
npm install -g vercel
```

#### 2. 登录 Vercel
```bash
vercel login
```
（会打开浏览器让你登录，可以用 GitHub 账号）

#### 3. 部署网站
在项目文件夹运行：
```bash
vercel
```

按照提示操作：
- Set up and deploy? → **Yes**
- Which scope? → 选择你的账号
- Link to existing project? → **No**
- What's your project's name? → 输入名字（比如 zm-studio）
- In which directory is your code located? → **./（直接回车）**
- Want to override the settings? → **No**

#### 4. 完成！
部署完成后会给你一个链接，类似：
```
https://zm-studio-xxxx.vercel.app
```

把这个链接发给你朋友就可以了！

#### 5. 后续更新
修改代码后，再次运行：
```bash
vercel --prod
```

## 方法二：Netlify 部署（也很简单）

### 在线部署（拖拽上传）：

1. 先构建网站：
```bash
npm run build
```

2. 访问 https://app.netlify.com/drop

3. 把 `dist` 文件夹直接拖进去

4. 立即获得网址！

### 命令行部署：

```bash
# 安装 Netlify CLI
npm install -g netlify-cli

# 登录
netlify login

# 部署
netlify deploy --prod --dir=dist
```

## 方法三：临时分享（ngrok - 快速演示）

如果只是临时展示几分钟/几小时：

1. 下载 ngrok：https://ngrok.com/download

2. 运行：
```bash
ngrok http 5173
```

3. 会给你一个临时链接，比如：
```
https://xxxx-xx-xxx.ngrok-free.app
```

**注意**：关闭电脑或停止命令后链接就失效了。

---

## 方法四：局域网分享（同一 WiFi）

如果你朋友在同一网络（同一 WiFi）：

```bash
npm run dev -- --host
```

然后你会看到：
```
➜  Local:   http://localhost:5173/
➜  Network: http://192.168.x.x:5173/
```

把 Network 那个地址发给朋友即可。

## ❓ 常见问题

**Q: 部署后图片不显示？**
A: 确保图片在 `public/images/` 文件夹里，然后重新部署。

**Q: 修改代码后怎么更新？**
A: Vercel - 运行 `vercel --prod`
   Netlify - 运行 `npm run build` 然后重新上传 `dist` 文件夹

**Q: 可以用自己的域名吗？**
A: 可以！在 Vercel/Netlify 后台添加自定义域名。

**Q: 要花钱吗？**
A: Vercel 和 Netlify 对个人项目完全免费！
